# Credits & Third-Party Assets

## 3D Models

All in-app 3D content (aircraft, world, and the cockpit interior) is generated
procedurally in code — no third-party model assets are bundled.

## Libraries

- **Three.js** r128 — MIT License — https://threejs.org (loaded from cdnjs CDN)
- **Post-processing passes** (EffectComposer, RenderPass, UnrealBloomPass,
  GammaCorrectionShader) — Three.js examples, MIT License (loaded from unpkg `three@0.128.0`)
- **node:test** — Node.js built-in test runner (Node 20+)
